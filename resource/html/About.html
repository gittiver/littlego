SgfcKit / libsgfc++ / SGFC

SgfcKit is the software library used to read and write SGF data. SgfcKit is an Objective-C wrapper library around the C++ library libsgfc++, which in turn is a wrapper for SGFC, the SGF Syntax Checker & Converter. SgfcKit and libsgfc++ are licensed under the Apache License, Version 2. SGFC is licensed under the BSD 3-Clause License. This version of %applicationName% uses SgfcKit 2.0.1, libsgfc++ 2.0.1 and SGFC 2.0
